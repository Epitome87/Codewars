# Function 2 - Squaring an Argument

## Description

Now you have to write a function that takes an argument and returns the square of it.

## My Solution

**JavaScript**

```js
const square = (n) => n ** 2;
const square = (n) => n * n;
const square = (n) => Math.pow(n, 2);
```

**TypeScript**

```ts
export const square = (num: number): number => num ** 2;
```
