# [Hex to Decimal](https://www.codewars.com/kata/57a4d500e298a7952100035d)

## Description

Complete the function which converts hex number (given as a string) to a decimal number.

## My Solution

**JavaScript**

```js
const hexToDec = (hexString) => parseInt(hexString, 16);
```

**Python**

```py
def hex_to_dec(s):
    return int(s, 16)
```
