# [Type of Sum](https://www.codewars.com/kata/5a2e9ae2b6cfd7692a0000ba)

## Description

Return the type of the sum of the two arguments

## My Solution

**JavaScript**

```js
const typeOfSum = (a, b) => typeof (a + b);
```

**TypeScript**

```ts
export function typeOfSum(a: any, b: any): string {
  return typeof (a + b);
}
```
