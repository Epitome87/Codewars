# [MakeUpperCase](https://www.codewars.com/kata/57a0556c7cb1f31ab3000ad7)

## Description

Write a function which converts the input string to uppercase.

## My Solution

**JavaScript**

```js
const makeUpperCase = (str) => str.toUpperCase();
```

**Python**

```py
def make_upper_case(s):
    return s.upper()
```
