# [Reverse List](https://www.codewars.com/kata/57a04da9e298a7ee43000111)

## Description

Write reverseList function that simply reverses lists.

## My Solution

**JavaScript**

```js
const reverseList = (arr) => [...arr].reverse();
```

**Python**

```py
def reverse_list(lst):
    return lst[::-1]
```
